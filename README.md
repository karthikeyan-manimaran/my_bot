# my_bot — Autonomous Mining Exploration (ROS2 Humble)

TurtleBot3 SLAM, manual teleop, assisted obstacle avoidance, and autonomous
frontier exploration in a custom underground mine world (Gazebo Classic 11).

## Run
- Manual mapping:        `python3 run_manual_mapping.py`
- Autonomous explore:    `python3 run_autonomous_explore.py`

Drive manually with arrow keys; press `s` to save the map, `q` to quit.
